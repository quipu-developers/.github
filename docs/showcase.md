# QUIPU Projects Showcase

## 1) QUIPU Main Web (2024)

### Overview
QUIPU의 2024 공식 웹사이트로, 동아리 소개부터 모집 안내, 활동 기록, 프로젝트 쇼케이스까지 하나의 흐름으로 제공한 메인 플랫폼입니다.

### Period
- 2024.01 ~ 2024.03 (Main Web)
- 2024.08 (Roulette Event)

### Tech Stack
- Frontend: React
- Backend: Express
- Database: MySQL

### Links
- Service: <https://quipu-main-2024.vercel.app/>
- GitHub: <https://github.com/quipu-uos/main-2024>

### Highlights
- 동아리 대외 안내와 활동 아카이브를 통합한 공식 웹 구성
- 모집, 소개, 활동, 쇼케이스 정보를 직관적으로 탐색 가능한 구조
- 하반기 이벤트 운영을 위한 룰렛 기능 기획·개발 및 실제 적용

## 2) 배틀글라운드 (2024)

### Overview
서울시립대학교 2024 인향제 축제 부스에서 운영된 실시간 멀티플레이 타자 게임입니다. 총 70명이 참여했으며, 현장형 경쟁 콘텐츠로 동아리 체험 몰입도를 높였습니다.

### Period
- 2024.07 ~ 2024.08

### Tech Stack
- Frontend: React
- Backend: Express, Socket.IO
- Database: MySQL

### Links
- GitHub: <https://github.com/quipu-uos/battleground>

### Highlights
- 방 생성/참가 기반 3인 실시간 멀티플레이 구조
- 60초 제한시간, 선점형 단어 입력 방식의 경쟁 룰
- 점수·순위·게임 상태의 실시간 동기화
- 종료 후 방 내 순위 및 전체 랭킹 TOP 10 제공

## 5) Articket

### Period
- 2024.09 ~ 2024.11 (Articket 1.0)
- 2025.02 (Articket 2.0)

### Tech Stack
- Frontend: React, WebSocket
- Backend: Flask, WebSocket
- AI/Infra: Stable Diffusion API, RunPod GPU, Railway, Netlify

### Links
- GitHub: <https://github.com/quipu-uos/Articket>

### Highlights
- 성격 테스트 기반 화가 스타일 추천(피카소, 르누아르, 고흐, 리히텐슈타인)
- 사용자 사진의 화풍 변환 및 비교 결과 제공
- 스마트폰-모니터 연동형 실시간 체험 플로우 구성
- 결과 이미지 기반 티켓 PDF 자동 생성·출력
